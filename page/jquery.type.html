$.type

jQuery.type method 로 Type 을 체크할 수 있다.
자세한 내용은 아래링크를 참조한다.

참조 : jQuery.type() | jQuery API Documentation 
		var str = "Hello World";
		
		var isStr = $.type(str) == 'string';		// true
		var isNum = $.type(str) == 'number';		// false
		var isObj = $.type(str) == 'object';		// false
		var isBool = $.type(str) == 'boolean';		// false
		var isFunc = $.type(str) == 'function';		// false
		//...
