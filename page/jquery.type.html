$.type

jQuery.type method 로 Type 을 체크할 수 있다.
자세한 내용은 아래링크를 참조한다.

참조 : jQuery.type() | jQuery API Documentation 
		var str = "Hello World";
		
		$.isString(str);		// true
		$.isNumber(str);		// false
		$.isObject(str);		// false
		$.isBoolean(str);		// false
		$.isFunction(str);		// false
		$.isArray(str);			// false
